# Changelog

这里记录 Halo 的公开版本变更。

## 0.3.116 - 2026-03-16

### 发布重点

- 完成 `0.3.116` Windows 桌面版打包，并验证安装包输出可用。
- 完成媒体运行时正式打包链路，确保点播播放所需的解析资源、运行时文件与播放器资源会进入发布包。
- 在公开推送前完成仓库清理，移除了本地研究资料、生成物、APK 分析资产与其他不应进入远端的内容。

### 新增

- 新增常驻解析进程路径，支持任务取消、诊断透传与运行时管理，用于点播解析。
- 新增桌面端点播兼容所需的 TVBox 运行时资源。
- 新增播放健康检查、传输处理、本地运行时桥接与结构化诊断相关模块。
- 新增面向发布的音乐模块与点播模块说明文档。

### 调整

- 调整桌面发布资源配置，正式纳入点播解析、播放器运行与兼容处理所需文件。
- 调整发布脚本和 bridge 构建脚本，只在必要时重建 bridge，避免开发态构建循环。
- 调整仓库收录边界，将 bridge 生成类、临时日志、APK 研究输出和其他可再生本地材料移出版本控制。
- 更新项目 README，使其反映当前模块结构与公开仓库边界。

### 修复

- 修复 bridge 构建中间产物写入 Tauri 监听目录后引发的持续重建问题。
- 修复点播切源时旧 spider 请求无法及时取消的问题，不再只依赖晚到结果丢弃。
- 提升点播解析链路可观测性，在播放成功与失败路径中附带结构化诊断信息。
- 完善发布包资源完整性，确保应用在正式版中能够正确找到解析文件、兼容资源与播放器运行文件。

### 模块说明

- 音乐模块：提供音乐播放、歌词获取、播放历史、迷你播放器与基础桌面交互能力，重点服务日常听歌和小窗控制场景。
- 点播模块：负责源导入、分类浏览、详情解析、播放链路调度与点播窗口体验，当前版本重点补强了解析稳定性、切源处理和正式版资源完整性。

### 验证

- 已验证 `pnpm exec tsc --noEmit`。
- 已验证 `cargo check`。
- 已验证 `cargo test spider_task_manager -- --nocapture`。
- 已验证 `pnpm exec vitest run src/modules/media/services/vodPlayback.resolve.test.ts`。
- 已验证 `pnpm bundle`，并产出 `0.3.116` 的 NSIS 安装包。

### 发布说明

- 当前 updater endpoint 仍指向现有局域网地址。如果这个版本后续要启用公开自动更新，需要单独切换为正式发布地址。
